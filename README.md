# Vendetta/Bunny based Plugins Template
This repo contains a template for creating plugins for Vendetta/Bunny based clients (e.g. Revenge, Kettu, etc).

# How to use?
To use this template, use GitHub's "Use this template" button to create a new repository. In the next screen, turn on "Include All Branches" and name the repository, then create your plugin repository.

# How to install?
Paste a plugin URL into the Plugins page of the client, following a basic format of:

https://`YOUR_GITHUB_USERNAME`.github.io/`REPO_NAME`/`PLUGIN_NAME`
